# Myreads Project
---

This is my first react project, Myreads.
Myreads is a web app which user can organize books with 3 categories:

- Currently Reading
- Want to Read
- Read


## Instalation
---

1. Clone this github repository
```
git clone https://github.com/hskang9/UdacityReactSubmission.git
```

2. Open app folder
```
cd reactnd-project-myreads
```

4. Install node packages 
```
npm install
```

5. Start react app 
```
npm start
```

6. Open in browser 
```
open http://localhost:3000
```
